Video File Metadata: Optimization For Google And YouTube

MAY 31, 2023 • BY PROBABLYRAGING

Often forgotten about, but probably one of the most important things you can do to get your videos ranked higher in Google and YouTube search results is the video file metadata.

 

What Is Video Metadata?

The metadata of a video file allows search engines to identify video files, making it easier to search, use and manage the video. The video metadata can, for instance, include a title for the video, a description, location, date of upload, and a bunch of other useful fields

 

Before You Upload Your Video

Right-click on your video file > properties > details. There are 3 fields here you need to edit. Title, Tags, and Comments

 

 

Title

This should be exactly the same as what you plan to title your video once you upload it. Try to add 3-5 relative keywords to your title as well

 

Tags

These are like YouTube's keywords, though, while keywords on YouTube aren't all that important, keywords in your video's metadata can help give search engines an idea of what your video is about. Include the keywords you've added to your title first and some other relative keywords. Try to add around 5-10 keywords, refrain from adding unrelated keywords as this could hurt your overall SEO

 

Comments

Is similar to YouTube's description section, but try to be briefer. Use about 10-20 words, again, using some of the keywords you have previously used. This can also be used in the YouTube description section but you should add more information and keywords

 

File Name

Once you have set your video's metadata, you should also change the file name from the generic "2022-08-17 18-09-46.mp4" to the same name you used in the title section
